# Описание лабораторной работы

Микросервис для создания заметок, их редактирования и удаления. Так же есть функционал выборки с пагинацией, поиска по тегам.


Подробная информация об эндпоинтах API в формате Swagger OpenAPI:
- `127.0.0.1:8000/docs` 

# Запуск

```
docker compose up --build
```

При запуске должен создаться инстанс микросервиса и PostgreSQL. Сразу же должны примениться миграции при запуске микросервиса, вроде как.
